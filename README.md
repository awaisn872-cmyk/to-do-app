# 📝 Todo App

A simple and responsive Todo App built using **HTML, CSS, and JavaScript**. This project helps users manage daily tasks with an easy-to-use interface and stores data using **Local Storage**, so tasks remain available even after refreshing the page.

## ✨ Features

* ➕ Add new tasks
* 📅 Select a due date
* ✅ Mark tasks as completed
* ↩️ Undo completed tasks
* 🗑️ Delete tasks
* 💾 Local Storage support
* ⌨️ Press **Enter** to add a task
* 📱 Responsive design
* 📆 Formatted dates (DD/MM/YYYY)

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 📖 How to Use

1. Enter your task.
2. Select a due date.
3. Click the **Add** button or press **Enter**.
4. Mark tasks as completed when finished.
5. Delete tasks whenever needed.
6. Your tasks are automatically saved in Local Storage.
